# 子技能：多势力冲突矩阵模板

> 用于设计多国家、多阵营、多暗线交织的剧情卷，避免所有势力都围着主角转。

## 必读文件

```text
00_settings/novel_settings.md
00_settings/world_map.json
03_characters/阵营与规则/相关阵营档案
03_characters/相关人物档案
02_memory/novel_directory.md
```

---

## 势力矩阵

```markdown
| 势力 | 公开目标 | 真实目标 | 掌握资源 | 不能失去什么 | 本卷行动 | 与主角交点 | 与其他势力冲突 |
|---|---|---|---|---|---|---|---|
|  |  |  |  |  |  |  |  |
```

---

## 暗线设计

每条暗线必须写清：

```markdown
### 暗线：{名称}

- 发起势力：
- 表面事件：
- 真实目的：
- 主角最初如何误解：
- 中段如何露出破绽：
- 卷末如何回收：
- 对下一卷的影响：
```

---

## 质量规则

1. 每个势力都必须有“不行动会失去什么”。
2. 阵营合作必须有临时利益，不默认天然同盟。
3. 阵营背刺必须有成本，不做随意反转。
4. 潮王、圣典院、商会、各国王权等大势力不能只做背景板。
5. 主角可以成为变量，但不能成为所有势力行动的唯一原因。
