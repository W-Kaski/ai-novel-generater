# 子技能：卷级大纲模板

> 用于设计、梳理、重修或归档 `02_memory/novel_directory.md` 中的卷级剧情。适合长篇小说的第 N 卷规划、卷内 100-200 章分段总结、多线交织、大场面和卷末承接。

## 必读文件

```text
00_settings/novel_settings.md
00_settings/character_state.md
00_settings/world_map.json
02_memory/novel_directory.md
02_memory/foreshadowing_ledger.md
02_memory/plot_arcs.md
03_characters/相关人物档案
03_characters/阵营与规则/相关阵营档案
```

---

## 核心原则

卷级大纲不得直接写事件列表。必须先确认：

- 主角为什么进入本卷地图。
- 主角赢了得到什么，输了失去什么。
- 每个重要人物为什么现在行动。
- 每个势力的公开目标和真实目标。
- 本卷新设定的权限、代价和边界。
- 本卷结尾如何自然引出下一卷。

---

## 输出结构

```markdown
【一、全局设定回顾】
1. 世界观/时代背景：
2. 力量体系/规则体系：
3. 主要阵营：
4. 重要地点：
5. 关键名词：
6. 主角当前身份、处境和长期目标：
7. 已知核心矛盾：

【二、第 {N} 卷总览】
用 5-8 句话概括：
- 本卷主线目标是什么
- 主角面临什么核心冲突
- 主要反派或阻碍是什么
- 本卷的关键转折是什么
- 本卷结尾把故事推向什么方向

【三、第 {N} 卷重要人物行动逻辑】
| 人物 | 本卷目标 | 主要行动 | 行动原因 | 结果/代价 | 后续影响 |
|---|---|---|---|---|---|

【四、第 {N} 卷逐章 / 分段总结】
第 X-Y 章：
- 事件概括：
- 人物行动逻辑：
- 结果/影响：
- 与下一段的衔接：

【五、第 {N} 卷伏笔与信息更新】
1. 新出现的设定：
2. 新出现的人物：
3. 新出现的矛盾：
4. 埋下的伏笔：
5. 尚未解决的问题：

【六、第 {N} 卷结尾效果】
1. 本卷主线是否完成：
2. 主角状态发生了什么变化：
3. 主要人物关系发生了什么变化：
4. 敌我局势发生了什么变化：
5. 哪些问题解决了：
6. 哪些问题留到下一卷：
7. 这个结尾如何引出第 {N+1} 卷：

【七、第 {N} 卷合理性锁定】
1. 为什么主角必须进入本卷地点：
2. 为什么本卷核心冲突现在爆发：
3. 为什么主要反派 / 阻碍不降智：
4. 为什么主角不能用更简单的方法解决：
5. 本卷新规则的权限边界：
6. 本卷战斗 / 大场面如何服务主题：
```

---

## 质量规则

1. 每个剧情单元必须先讲动机，再讲事件。
2. 不使用“突然发现”“被迫卷入”“卡住”这类模糊推进词。
3. 不让所有势力围着主角转；势力必须有自己的利益和行动逻辑。
4. 大场面必须有明确目标：抢什么、救谁、阻止什么、证明什么。
5. 本卷新概念必须能用一句人话解释，并写清权限边界。
6. 卷末必须完成一个阶段问题，同时留下更高层问题。
