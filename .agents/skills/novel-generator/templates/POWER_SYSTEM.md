# 子技能：力量体系、功法、特性与等级模板

> 用于维护 `00_settings/novel_settings.md` 中的世界规则与力量系统。

## 职责

写入 `novel_settings.md` 的详细世界规则与力量体系分组：

- 力量来源
- 等级 / 境界 / 阶段
- 功法或技术分类
- 特性 / 权柄 / 异能规则
- 战力边界
- 使用代价
- 失控风险
- 禁忌与例外
- 规则如何服务主题和剧情

角色独有能力写入角色档案；章节推进后的临时变化写入 `character_state.md`。

## 质量规则

1. 每个强能力必须有代价、限制或反制方式。
2. 等级差距不能只靠数字描述，要有可写进场景的表现差异。
3. 主角特殊机制必须既能解决问题，也能制造新问题。
4. 新例外必须有伏笔或代价，不能为单章便利凭空添加。
5. 改变战力规则后，检查已写章节是否被推翻。
