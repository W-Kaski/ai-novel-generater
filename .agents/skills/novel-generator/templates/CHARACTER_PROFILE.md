# 子技能：角色档案模板

> 用于维护 `03_characters/主要角色/*.md` 与 `03_characters/重要配角/*.md`。本文件记录静态角色档案，不记录逐章动态状态。

## 职责边界

角色档案负责：

- 出身与身份
- 核心欲望
- 恐惧、缺陷、执念
- 长期人物弧光
- 固定能力与能力边界
- 关系定位
- 叙事功能
- 可复用口吻与行为逻辑

不负责：

- 当前伤势
- 当前所在位置
- 当章即时目标
- 本章刚获得的信息

这些动态内容写入 `00_settings/character_state.md`。

## 更新规则

1. 角色长期设定改变时，先说明影响范围。
2. 如果只是章节推进后的状态变化，不修改角色档案，改 `character_state.md`。
3. 角色获得新长期能力时，同步检查 `novel_settings.md` 的力量体系是否允许。
4. 角色加入或背叛势力时，同步检查阵营档案与人物轨。
