# 子技能：小说设定总表模板

> 用于维护 `00_settings/novel_settings.md`。本文件承载作品级总设定、世界规则、历史真相、制度细则、力量体系与创作约束；不承载详细人物百科、地点百科或逐章动态状态。

## 文件职责

> 架构迁移说明：旧版本拆出的独立架构设定文件已废弃，世界观长文、历史真相、制度细节和力量体系扩展说明均迁移到 `00_settings/novel_settings.md` 的分层章节中维护。

| 文件 / 位置 | 负责内容 | 不负责内容 |
| --- | --- | --- |
| `00_settings/novel_settings.md` | 作品级总设定、故事引擎、世界规则、历史真相、制度细节、力量体系扩展说明、结构规划、地图/势力索引、创作约束 | 详细角色小传、详细城市百科、逐章动态状态 |
| `03_characters/` | 角色档案、阵营档案、关系定位、角色个人弧光 | 作品级总目标和总叙事结构 |
| `00_settings/world_map.json` | 地图实体、地点坐标、势力分布、地图可视化数据 | 长篇文字版角色介绍 |
| `00_settings/character_state.md` | 当前阶段角色动态变化：位置、伤势、关系、心理、已知信息 | 角色静态设定总表 |
| `02_memory/novel_directory.md` | 章节级目录、大纲、每章任务 | 世界观总论 |

规则：

1. `novel_settings.md` 是唯一设定总入口，可分层记录上层结论与详细规则。
2. 角色详细资料只写进 `03_characters/`。
3. 地图、城市、势力的详细展开以 `world_map.json` 与阵营档案为准。
4. `novel_settings.md` 可以保留角色、地点、势力索引，但不要复制整份角色卡或地点百科。
