# 子技能：角色当前状态模板

> 本文件对应主技能 SKILL.md §11.3，用于更新和维护 `character_state.md` 中的角色动态状态卡片。

## 用途

当 Agent 需要新增、更新某个角色在章节推进后的动态状态时，加载本文件获取模板规范。  
状态文件路径：`{project}/00_settings/character_state.md`

---

## 职责边界

`character_state.md` **只记录动态状态**，不承担静态角色百科功能：

| 应写入 character_state.md  | 应写入 `03_characters/` |
| -------------------------- | ------------------------------ |
| 当前位置                   | 背景故事、出身                 |
| 当前目标（即时目标）       | 外貌、穿着描述                 |
| 当前情绪 / 心理状态        | 静态人物关系图                 |
| 当前身体状态（伤势等）     | 能力全集、技能列表             |
| 当前掌握的信息             | 长期目标、信仰、价值观         |
| 当前关系变化（本章新变动） | 固定设定百科                   |
| 下一步行动倾向             |                                |

---

## 模板格式

```markdown
# {角色姓名} · 当前状态

## 1. 当前所在位置【必填】

角色现在在哪里？是否能立刻参与下一章剧情？

填写示例：
当前位于清风观后山丹房，被丹阳子安排继续处理药材，暂时无法自由离开。

---

## 2. 当前身份处境【必填】

角色现在处于什么身份、地位、危险处境中？

填写示例：
表面是丹阳子的弟子，实际已经成为重点观察对象，随时可能被当作炼丹材料。

---

## 3. 当前目标【必填】

角色眼下最想完成什么，不是长期目标。

填写示例：
暂时隐藏自己已经发现丹房异常的事实，寻找逃离丹房的机会。

---

## 4. 当前情绪 / 心理状态【推荐】

角色此刻的精神状态，影响下一章语气和行为。

填写示例：
恐惧、警觉、强行冷静，对丹阳子产生强烈戒备。

---

## 5. 当前身体状态【必填】

伤势、疲劳、污染、修为、异常状态都写这里。

填写示例：
身体疲惫，右手被药炉烫伤，精神受到丹房幻觉轻微影响。

---

## 6. 当前掌握的信息【必填】

角色现在知道什么，尤其是秘密、误解、线索。

填写示例：
知道丹房药材可能来自活人；怀疑丹阳子并非真正传道，而是在筛选"药材"。

---

## 7. 当前关系变化【必填】

记录这个角色和其他关键角色的关系状态（本章新发生的变化）。

填写示例：

- 对丹阳子：表面顺从，内心恐惧和戒备。
- 对其他弟子：保持距离，不敢透露自己发现的异常。
- 对主角团：暂无可信任对象。

---

## 8. 下一步行动倾向【必填】

AI 写下一章时，这个角色自然会做什么。

填写示例：
继续伪装顺从，暗中观察丹房出口、药童作息和丹阳子的试探规律。
```

---

## 填写规则

1. 字段 1、2、3、5、6、7、8 为**必填**。
2. 字段 4（情绪状态）为推荐，建议每章更新，用于保持文风一致性。
3. 每章定稿后，Agent **必须**更新所有出场角色的状态卡。
4. 若某章角色未出场但有状态变化（如收到情报、被远程行动影响），也应更新并注明来源。
5. 不得用此文件替代 `03_characters/`；不得把能力全集、背景故事、外貌写入此文件。
