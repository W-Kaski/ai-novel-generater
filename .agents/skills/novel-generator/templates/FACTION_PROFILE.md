# 子技能：势力与阵营档案模板

> 用于维护 `03_characters/阵营与规则/*.md`。

## 职责

阵营档案负责：

- 势力目标
- 组织结构
- 资源与控制范围
- 公开秩序与隐藏秩序
- 内部矛盾
- 与主角和其他势力的关系
- 可持续制造的剧情冲突

地图坐标、路线和空间控制范围写入 `00_settings/world_map.json`；这里记录制度、利益和叙事功能。

---

## 推荐路径

```text
03_characters/阵营与规则/{势力名}.md
```

## 同步规则

1. 改变势力控制区域时，同步 `world_map.json`。
2. 改变势力核心规则时，同步 `novel_settings.md`。
3. 改变势力与主角关系时，同步 `plot_arcs.json` 与 `character_state.md`。
4. 在 `novel_settings.md` 中只保留势力索引。
